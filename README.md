# Loan Prediction ML Project

This project uses Machine Learning to predict loan approvals based on applicant data.

## Tech Stack
- **Python**
- **Scikit-learn**
- **Pandas**
- **Numpy**
- **Matplotlib (Optional for visualization)**

## Features
- Predicts loan approval based on features such as:
  - Gender
  - Marital Status
  - Applicant's Income
  - Loan Amount
  - Credit History
  - Property Area
  - and more...
